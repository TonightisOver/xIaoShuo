3. 典型请求处理流程 示例：新建人物

sequenceDiagram
    actor User as 前端用户
    participant Router as FastAPI Router
    participant Route as characters.py
    participant Service as character_service.py
    participant DB as PostgreSQL

    User->>Router: POST /api/v1/projects/{id}/characters
    Router->>Route: 匹配路由
    Route->>Route: 解析 JSON → CharacterRequest
    Route->>Service: get_character_service().create_character(data)
    Service->>Service: 校验参数
    Service->>DB: INSERT INTO characters ...
    DB-->>Service: 返回 ORM 对象
    Service-->>Route: 返回 dict
    Route->>Route: 序列化为 JSONResponse
    Route-->>User: 201 Created + 人物数据
