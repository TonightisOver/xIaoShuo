物理沙箱级环境隔离 System Layer

利用底层 Antigravity Workspace 调度层。在调用 Subagent 时支持三种模式（inherit, branch, share），可在不破坏开发工作区的前提下，并发克隆独立工作目录作为沙箱，保障代码安全。

防污染机制 · 零溢出 Isolation Rate: 100%

编排状态机级数据继承 Application Layer

由主编排器 Orchestrator 通过流程网闸控制，从物理工作区统一读取变更目录下的历史文档（如需求分析、技术设计），并以摘要和指令拼接的形式级联注入下流子智能体 Prompt，实现业务链条完整传承。

门禁审批机制 · 级联依赖 Data Lineage: Strict
