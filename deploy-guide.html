5修改服务器密码（重要！）

⚠️ 安全提醒：你的服务器 root 密码和 GitHub token 已在对话中暴露。部署完成后请：
1. 修改服务器密码
2. 在 GitHub Settings → Developer settings → Personal access tokens 中删除该 token 并重新生成

如果遇到问题

查看容器日志：

docker-compose logs api    # 查看 API 日志
docker-compose logs db     # 查看数据库日志
docker-compose logs nginx  # 查看 Nginx 日志

重启所有服务：

docker-compose restart

完全重建：

docker-compose down
docker-compose up -d --build
