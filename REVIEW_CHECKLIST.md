## Review 流程

### 阶段 1: 创建小说项目

1. 打开 http://115.190.142.169
2. 点击"开始创作"
3. 填写：
   - 标题：测试小说
   - 创意：一个现代程序员穿越到修仙世界，用编程思维修炼
   - 类型：玄幻
   - 文风：选择"自定义"，输入"江南风格，带灰色幽默"，点击"生成风格指令"
   - 字数：10万
4. 点击"开始生成"

**检查项：**
- [ ] 创建成功，跳转到小说详情页
- [ ] 文风选择 UI 可见且可用
- [ ] 自定义文风生成指令成功

---

### 阶段 2: 人机对话

1. 进入小说详情页 → "创作对话" Tab
2. 点击"新建对话"，主题输入"讨论世界观和人物设定"
3. 发送消息："我想要一个以灵气为基础的修仙世界，有九大境界"
4. 等待 AI 回复
5. 在 AI 回复上点击"确定为世界观"

**检查项：**
- [ ] 对话创建成功
- [ ] AI 回复正常（引用了小说设定）
- [ ] "确定为世界观/人物/故事线"按钮可见
- [ ] 点击"确定为世界观"后消息标记为"已确定"
- [ ] 世界观页面能看到刚确定的内容

---

### 阶段 3: 继续对话并确定人物

1. 继续发送："主角叫林墨，是一个程序员，性格沉稳"
2. AI 回复后点击"确定为人物"
3. 点击"结束对话"
4. 查看结论区域

**检查项：**
- [ ] 人物确定成功
- [ ] "人物" Tab 能看到新创建的人物
- [ ] 结束对话后显示结论
- [ ] "从对话生成故事线"按钮可用
- [ ] "生成总纲→卷纲"按钮可用

---

### 阶段 4: 生成故事线

1. 点击"从对话生成故事线"
2. 或进入"故事线" Tab → "管理故事线" → 点击"AI 生成"

**检查项：**
- [ ] 故事线生成成功（有主线/副线）
- [ ] "AI 生成"按钮可用（故事线/弧光/场景）
- [ ] 生成的内容与小说设定相关

---

### 阶段 5: 大纲体系

1. 进入"大纲" Tab → 点击"编辑大纲"
2. 如果总纲为空，手动填写或点击对话页的"生成总纲→卷纲"
3. 填写总纲后点击"保存"
4. 点击"生成卷纲"
5. 等待卷纲生成
6. 选择卷一，点击"生成章纲"
7. 等待章纲生成

**检查项：**
- [ ] 总纲可编辑并保存
- [ ] "生成卷纲"按钮可用
- [ ] 卷纲正确显示（多卷）
- [ ] "生成章纲"按钮可用
- [ ] 章纲只显示在对应卷下（不是全部混在一起）
- [ ] 每卷下方有"生成本卷章节"按钮

---

### 阶段 6: 生成章节

1. 在大纲编辑器中，点击卷一的"生成本卷章节"
2. 或在章节 Tab 中点击"按范围生成"

**检查项：**
- [ ] "生成本卷章节"触发成功
- [ ] 跳转到任务详情页
- [ ] 任务详情页显示进度（或"准备中"）
- [ ] 生成完成后可查看章节
- [ ] "按范围生成"弹窗可用

---

### 阶段 7: 章节管理

1. 进入章节 Tab，点击某一章进入编辑
2. 修改内容，点击"保存"
3. 点击"重新生成"
4. 返回章节列表，尝试删除某章

**检查项：**
- [ ] 章节编辑页可正常编辑和保存
- [ ] "重新生成"按钮可用（跳转到任务页）
- [ ] "删除"按钮可用（确认后删除）
- [ ] 字数实时统计

---

### 阶段 8: 世界观/人物编辑

1. 进入"世界观" Tab → 点击"编辑"
2. 修改世界观内容，保存
3. 进入"人物" Tab → 添加新人物 → 删除人物

**检查项：**
- [ ] 世界观编辑页可保存
- [ ] 人物可添加/删除
- [ ] 保存后返回详情页数据更新

---

### 阶段 9: 图谱可视化

1. 进入"故事线" Tab → 点击"查看图谱"
2. 查看树形图

**检查项：**
- [ ] 图谱页面加载成功
- [ ] 树形图正确显示（故事线/人物弧光/场景分支）
- [ ] 节点颜色区分正确

---

### 阶段 10: 删除小说

1. 返回书架首页
2. 点击某本小说的"删除"按钮

**检查项：**
- [ ] 删除确认弹窗出现
- [ ] 确认后小说从列表消失
- [ ] 关联数据（章节/设定/对话）一并删除

## API 端点快速验证

```bash

curl http://localhost/api/v1/health

# 项目列表
curl http://localhost/api/v1/projects

# 创建项目（需要有效的请求体）
curl -X POST http://localhost/api/v1/projects \
  -H "Content-Type: application/json" \
  -d '{"idea":"test idea for review","novel_type":"玄幻","target_words":100000,"title":"Review Test"}'

# 大纲树
curl http://localhost/api/v1/projects/{novel_id}/outlines

# 关系图谱数据
curl http://localhost/api/v1/projects/{novel_id}/relations

# 对话列表
curl http://localhost/api/v1/projects/{novel_id}/conversations
```
