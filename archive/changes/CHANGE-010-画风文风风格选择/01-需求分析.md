# 需求分析: 画风/文风风格选择

**变更 ID**: CHANGE-010  
**创建时间**: 2026-05-16  
**优先级**: P1

## 1. 需求背景

用户希望在创建小说时能选择写作风格（文风），让 AI 生成的内容具有不同的文学特色。例如同样是玄幻题材，可以选择"轻松幽默"或"史诗厚重"的风格，产出截然不同的文本。

**目标用户**: 小说创作者  
**使用场景**: 创建小说项目时选择风格，影响所有后续生成内容

## 2. 核心功能点

### 功能 1: 文风风格定义
**描述**: 预设一组文风选项供用户选择  
**优先级**: P0

**预设风格**:

| 风格 | 说明 | 特点 |
|------|------|------|
| 轻松幽默 | 诙谐搞笑，节奏轻快 | 吐槽、梗、轻松对话 |
| 热血燃向 | 激情澎湃，战斗爽快 | 短句、力量感、节奏紧凑 |
| 细腻文艺 | 文笔优美，注重描写 | 长句、意象、心理描写 |
| 史诗厚重 | 宏大叙事，气势磅礴 | 正式用语、宏观视角 |
| 悬疑紧张 | 氛围营造，节奏紧凑 | 短段落、悬念、伏笔 |
| 古风典雅 | 半文言风格，古韵十足 | 四字词、诗词引用 |
| 现代白话 | 通俗易懂，贴近生活 | 口语化、生活化 |
| 暗黑压抑 | 阴暗沉重，心理深度 | 内心独白、环境渲染 |

### 功能 2: 风格影响生成
**描述**: 选择的风格作为 prompt 指令注入到 LLM 调用中  
**优先级**: P0  
**实现**: 在每个生成节点的 prompt 中添加风格描述

### 功能 3: 前端风格选择 UI
**描述**: 创建小说时可选择风格  
**优先级**: P0  
**位置**: Create.vue 表单中，类型选择下方

### 功能 4: 风格持久化
**描述**: 风格保存在 novels 表中  
**优先级**: P0  
**字段**: `writing_style VARCHAR(50)`

## 3. 非功能需求

- 风格选择可选（默认"现代白话"）
- 风格可在小说项目中修改（影响后续生成）
- 不影响已生成的章节内容

## 4. 验收标准

- [ ] 创建小说时可选择 8 种文风
- [ ] 选择的风格保存到数据库
- [ ] 生成时 prompt 包含风格指令
- [ ] 小说详情页显示当前风格
- [ ] 可修改风格设置
- [ ] 所有测试通过
