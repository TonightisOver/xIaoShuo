## Bug 修复详情

### Bug 1: 章节生成脱离上下文（根因修复）
- **根因**: `generate_volume_background()` 和 `generate_chapters_background()` 硬编码 `characters=json.dumps([])` 和 `world_setting=json.dumps({})`
- **修复**: 生成前从数据库获取真实人物/世界观/故事线，构建完整 prompt 上下文

### Bug 2: 卷纲跨表问题
- **根因**: OutlineEditor 生成卷纲存在 `outlines` 表，`generate_volume_background` 只查 `volumes` 表
- **修复**: 新增 outlines 表 fallback，两个表都查不到才报错

### Bug 3: 人物弧光脱离上下文
- **根因**: `generate_arcs_ai()` 只传角色名+角色类型给 LLM
- **修复**: 注入小说类型/idea/世界观/已有故事线/总纲

### Bug 4: 按范围生成章节失效
- **根因**: 与 Bug 1 同源 — 空人物/世界观导致生成质量极差
- **修复**: 同 Bug 1

### Bug 5: 前端概览标签
- **修复**: "创意" → "简介"，长文本支持换行

## 子资源 API 一致性修复

给 14 个 `GET/POST/PUT/DELETE` 子资源端点添加了项目存在性校验，删除后统一返回 404。
