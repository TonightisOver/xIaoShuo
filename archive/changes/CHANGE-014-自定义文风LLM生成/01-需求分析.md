# 需求分析: 自定义文风（LLM 生成）

**变更 ID**: CHANGE-014  
**创建时间**: 2026-05-16  
**优先级**: P1

## 1. 需求背景

CHANGE-010 实现了 8 种预设文风选择。但用户希望能用自然语言描述更个性化的风格，例如：
- "使用江南的文风，带一点灰色幽默"
- "模仿金庸的武侠叙事，但更现代化"
- "像东野圭吾那样的悬疑节奏，但加入中国元素"

需要通过 LLM 将用户的自然语言描述转化为具体的风格指令，注入到生成 prompt 中。

**目标用户**: 对文风有明确偏好的创作者  
**使用场景**: 创建小说时选择"自定义"风格，输入自然语言描述

## 2. 核心功能点

### 功能 1: 自定义风格输入
**描述**: 在预设 8 种风格之外，添加"自定义"选项，用户输入自然语言描述  
**优先级**: P0  
**UI**: 选择"自定义"后展开文本输入框

### 功能 2: LLM 风格指令生成
**描述**: 调用 LLM 将用户描述转化为具体的写作风格指令  
**优先级**: P0  
**流程**: 用户输入 → 调用 LLM → 生成风格指令 → 保存到 novels.writing_style_prompt

### 功能 3: 风格预览
**描述**: 生成风格指令后展示给用户确认  
**优先级**: P1  
**UI**: 显示 LLM 生成的风格指令，用户可编辑或重新生成

## 3. 数据模型变更

### novels 表新增字段

```sql
ALTER TABLE novels ADD COLUMN custom_style_description TEXT;
ALTER TABLE novels ADD COLUMN writing_style_prompt TEXT;
```

- `custom_style_description`: 用户原始输入（如"江南风格+灰色幽默"）
- `writing_style_prompt`: LLM 生成的具体风格指令（注入到生成 prompt 中）

### 逻辑

- 如果 `writing_style` 是预设 8 种之一 → 使用 WRITING_STYLES 常量
- 如果 `writing_style` = "自定义" → 使用 `writing_style_prompt` 字段

## 4. LLM 风格生成 Prompt

```
你是一位文学风格分析专家。用户描述了他们想要的写作风格，请将其转化为一段具体的写作指令（100-200字），用于指导 AI 生成小说内容。

用户描述：{user_description}

请输出一段具体的风格指令，包含：
- 句式特点（长短句、节奏）
- 用词风格（文雅/通俗/专业）
- 叙事视角和语气
- 特殊技巧（比喻手法、对话风格等）

只输出风格指令，不要其他说明。
```
