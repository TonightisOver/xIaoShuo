# 变更总结: 人机交互问答生成

**变更 ID**: CHANGE-013  
**状态**: 已完成  
**完成时间**: 2026-05-16

## 变更概述

为小说创作平台新增"创作对话"功能，允许用户与 AI 进行多轮对话讨论小说设定和情节，对话结束后自动提取结构化建议。

## 交付物

| 组件 | 内容 |
|------|------|
| 数据库 | conversations + messages 两张新表，Alembic 迁移 |
| 后端服务 | conversation_service.py — 对话管理 + LLM 调用 + 结论提取 |
| API 路由 | conversations.py — 5 个端点（CRUD + 发消息 + 结束） |
| 前端页面 | Conversation.vue（聊天 UI）+ NovelDetail.vue（对话入口） |

## 关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 回复方式 | 同步 | 简单可靠，后续可升级 SSE |
| 上下文窗口 | 最近 20 轮 | 平衡质量与 token 消耗 |
| 结论提取 | LLM 结构化输出 | 自动化程度高 |
| 前端方案 | 聊天气泡 + Tab 集成 | 入口自然，交互直观 |
