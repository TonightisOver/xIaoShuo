## 使用示例

### 基础使用

```python
import asyncio
from src.core.langgraph.graph import create_novel_graph
from src.core.langgraph.state import NovelState

async def main():
    # 创建工作流
    graph = create_novel_graph()
    
    # 初始化状态
    initial_state: NovelState = {
        "project_id": "novel-001",
        "idea": "一个程序员穿越到修仙世界",
        "novel_type": "玄幻",
        "target_words": 100000,
        "current_stage": "start",
        "chapters": [],
        "errors": [],
    }
    
    # 执行工作流
    result = await graph.ainvoke(initial_state)
    
    # 查看结果
    print(f"生成章节数: {len(result['chapters'])}")
    print(f"当前阶段: {result['current_stage']}")

if __name__ == "__main__":
    asyncio.run(main())
```

- 磁盘空间 < 1GB → 告警

### 常见问题

#### 1. API 密钥无效

**症状**: `AuthenticationError: Your api key is invalid`

**解决**:
1. 检查 `.env` 文件中的 `DEEPSEEK_API_KEY`
2. 访问 https://platform.deepseek.com 验证密钥
3. 清除环境变量: `$env:DEEPSEEK_API_KEY = $null`

#### 2. API 超时

**症状**: `Request timed out`

**解决**:
1. 增加 `DEEPSEEK_TIMEOUT` (默认 120 秒)
2. 检查网络连接
3. 简化 prompt 长度

#### 3. JSON 解析失败

**症状**: 日志中出现 JSON 解析错误

**解决**:
- 系统会自动使用 fallback 数据
- 检查日志中的原始响应
- 如果频繁发生，考虑优化 prompt
