## 数据模型

### 新增表
| 表名 | 主要字段 | 外键 |
|------|----------|------|
| novels | id, title, description, genre, status | - |
| world_settings | id, novel_id, setting_type, content | novels.id |
| power_systems | id, novel_id, name, rules, levels | novels.id |
| characters | id, novel_id, name, role, backstory | novels.id |
| chapters | id, novel_id, chapter_number, title, content | novels.id |

所有子表对 novel_id 设置 CASCADE 删除。

### 模型关系
- Novel 1:N WorldSetting
- Novel 1:N PowerSystem
- Novel 1:N Character
- Novel 1:N Chapter
- Task N:1 Novel (可选关联)

## API 设计

基础路径: `/api/v1/projects`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | / | 创建项目 |
| GET | / | 列表查询 |
| GET | /{id} | 获取详情 |
| PUT | /{id} | 更新项目 |
| DELETE | /{id} | 删除项目(级联) |
| POST | /{id}/world | 创建世界设定 |
| GET | /{id}/world | 获取世界设定 |
| PUT | /{id}/world/{wid} | 更新世界设定 |
| POST | /{id}/characters | 创建角色 |
| GET | /{id}/characters | 角色列表 |
| PUT | /{id}/characters/{cid} | 更新角色 |
| DELETE | /{id}/characters/{cid} | 删除角色 |
| POST | /{id}/power-systems | 创建力量体系 |
| GET | /{id}/power-systems | 力量体系列表 |
| PUT | /{id}/power-systems/{pid} | 更新力量体系 |
| DELETE | /{id}/power-systems/{pid} | 删除力量体系 |
| POST | /{id}/chapters | 创建章节 |
| GET | /{id}/chapters | 章节列表 |
| GET | /{id}/chapters/{chid} | 章节详情 |
| PUT | /{id}/chapters/{chid} | 更新章节 |
| DELETE | /{id}/chapters/{chid} | 删除章节 |

## 服务层设计

`NovelManager` 类职责:
- CRUD novels
- CRUD world_settings (按 novel 过滤)
- CRUD power_systems (按 novel 过滤)
- CRUD characters (按 novel 过滤)
- CRUD chapters (按 novel 过滤)
