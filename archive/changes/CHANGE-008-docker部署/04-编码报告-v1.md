## 关键实现细节

### Dockerfile
- Stage 1: `node:20-slim` 执行 `npm ci && npm run build`
- Stage 2: `python:3.14-slim` 安装 Python 依赖，复制前端产物到 static 目录
- 入口: `alembic upgrade head && uvicorn main:app --host 0.0.0.0 --port 8000`

### docker-compose.yml
- db: `postgres:17-alpine`，healthcheck 使用 `pg_isready`
- api: depends_on db (condition: service_healthy)
- nginx: depends_on api

### deploy.sh
- 检测并安装 Docker + Docker Compose
- 拉取代码 / 复制文件
- `docker-compose up -d --build`
