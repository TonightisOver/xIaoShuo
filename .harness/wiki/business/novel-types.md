# 小说类型与设定体系

## 支持的小说类型

| 类型 | 说明 | 典型设定 |
|------|------|---------|
| 玄幻 | 东方玄幻世界 | 修炼体系、宗门、灵气 |
| 仙侠 | 修仙求道 | 境界划分、法宝、飞升 |
| 都市 | 现代都市 | 商战、异能、重生 |
| 科幻 | 未来科技 | 星际、AI、时间旅行 |
| 历史 | 历史架空 | 朝代、权谋、战争 |
| 武侠 | 江湖武林 | 武功、门派、江湖恩怨 |
| 言情 | 情感故事 | 人物关系、情感冲突 |
| 悬疑 | 推理悬疑 | 案件、线索、反转 |
| 军事 | 军事题材 | 战役、军队、战略 |
| 同人 | 同人创作 | 基于已有作品 |
| 游戏 | 游戏世界 | 等级、副本、技能树 |
| 灵异 | 灵异恐怖 | 鬼怪、驱邪、阴阳 |
| 竞技 | 竞技体育 | 比赛、训练、成长 |

## 设定体系结构

每本小说项目包含以下设定维度：

### 世界观 (world_settings)

- **世界背景**: 世界的基本设定和历史
- **地理环境**: 大陆、国家、地形、特殊地点
- **文化体系**: 宗门/势力/社会结构/信仰
- **世界规则**: 力量法则、修炼原理、限制条件

### 等级体系 (power_systems)

一本小说可以有多个力量体系，每个体系包含：
- **名称**: 如"修仙境界"、"斗气等级"
- **描述**: 体系的基本原理
- **等级列表**: 从低到高的等级划分，每级有名称、描述、突破条件

示例：
```json
{
  "name": "修仙境界",
  "levels": [
    {"name": "练气期", "description": "感知灵气，引气入体"},
    {"name": "筑基期", "description": "凝聚灵力，构建根基"},
    {"name": "金丹期", "description": "灵力凝丹，质变飞跃"},
    {"name": "元婴期", "description": "元神出窍，分身化形"}
  ]
}
```

### 人物 (characters)

- **姓名**: 角色名
- **角色定位**: 主角/配角/反派/导师
- **描述**: 外貌、身份
- **性格**: 性格特点
- **能力**: 特殊技能/武功/法术
- **背景故事**: 人物经历

### 章节 (chapters)

- **章节号**: 顺序编号
- **标题**: 章节标题
- **内容**: 正文（可编辑）
- **字数**: 自动统计
- **状态**: draft/generated/edited
