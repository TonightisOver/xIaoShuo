# 数据库设计规范

**版本**: v1.0  
**适用项目**: xIaoShuo - AI 中文网络小说生成平台  
**数据库**: PostgreSQL 14+  
**ORM**: SQLAlchemy 2.0+

---

## 1. 表命名规范

### 1.1 命名风格

- **使用小写 + 下划线**（snake_case）
- **使用复数形式**（表示集合）
- **避免使用保留字**

```sql
-- ✅ 正确
CREATE TABLE projects (...);
CREATE TABLE novel_chapters (...);

-- ❌ 错误
CREATE TABLE Project (...);  -- 大写
CREATE TABLE project (...);  -- 单数
```

### 2.1 常用字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `id` | UUID | 主键 |
| `created_at` | TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | 更新时间 |

## 3. 数据类型选择

### 3.1 主键

**使用 UUID**：

```sql
id UUID PRIMARY KEY DEFAULT gen_random_uuid()
```

### 3.2 字符串

```sql
name VARCHAR(255)  -- 名称
description TEXT  -- 描述
```

### 3.3 日期时间

```sql
created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
```

## 4. 索引

### 4.1 创建索引

```sql
-- 单列索引
CREATE INDEX idx_projects_user_id ON projects(user_id);

-- 复合索引
CREATE INDEX idx_chapters_project_number ON chapters(project_id, chapter_number);
```

---

## 5. SQLAlchemy 模型

### 5.1 基础模型

```python
from sqlalchemy import Column, DateTime
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.dialects.postgresql import UUID
import uuid

Base = declarative_base()

class BaseModel(Base):
    __abstract__ = True
    
    id = Column(UUID(as_uuid=True), primary_key=True, default=uuid.uuid4)
    created_at = Column(DateTime, nullable=False)
    updated_at = Column(DateTime, nullable=False)
```

## 6. 最佳实践

### ✅ 推荐做法

1. **使用 UUID 作为主键**
2. **使用 TIMESTAMP WITH TIME ZONE 存储时间**
3. **为外键创建索引**
4. **使用 Alembic 管理数据库迁移**
5. **避免 N+1 查询**

### ❌ 避免做法

1. **不要使用自增 ID 作为主键**
2. **不要忽略索引**
3. **不要在事务中执行耗时操作**
