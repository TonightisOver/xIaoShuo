# 需求分析

## 变更编号
CHANGE-006

## 变更标题
前端界面

## 需求概述
为小说生成平台添加 Vue 3 前端界面，提供任务管理、小说详情、世界观编辑、角色管理、章节编辑等功能的可视化操作入口。

## 功能需求

### 页面列表
| 页面 | 路由 | 说明 |
|------|------|------|
| Home | / | 首页，展示任务卡片网格 |
| Create | /create | 创建新任务 |
| TaskDetail | /task/:id | 任务详情与进度 |
| NovelDetail | /novel/:id | 小说详情展示 |
| WorldEdit | /world/:id | 世界观编辑 |
| Characters | /characters/:id | 角色管理 |
| ChapterEdit | /chapter/:id | 章节编辑 |

### 公共组件
- TaskCard: 任务卡片，含状态徽章
- ProgressBar: 进度条
- StageIndicator: 阶段指示器

### 实时通信
- WebSocket composable (useWebSocket.js) 用于任务进度实时推送

## 非功能需求
- 响应式布局，适配桌面与移动端
- 构建产物: JS ~95KB + CSS ~17KB (gzipped)
- 单端口部署，嵌入 FastAPI 静态文件服务
- SPA fallback 路由支持
- 字体: Noto Sans SC / Noto Serif SC

## 设计风格
参考起点/番茄小说，但更精致简约。卡片式布局、状态徽章、响应式网格。

## 验收标准
- 所有 7 个页面可正常路由访问
- WebSocket 连接正常建立
- 13 项测试全部通过
- 构建产物体积符合预期
