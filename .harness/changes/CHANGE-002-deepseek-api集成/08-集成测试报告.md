## 发现的问题

### 🟡 中优先级

1. **API 超时配置过短**
   - 问题: 默认 30 秒超时不足以处理复杂 prompt
   - 影响: 世界观构建等复杂节点会超时
   - 解决: 增加超时到 120 秒
   - 状态: ✅ 已修复

2. **max_tokens 参数警告**
   - 问题: `UserWarning: Parameters {'max_tokens'} should be specified explicitly`
   - 影响: 仅警告，不影响功能
   - 解决: 使用 `model_kwargs` 传递参数
   - 状态: ✅ 已处理（使用 type: ignore）

### 🟢 低优先级

1. **Deprecation 警告**
   - 问题: Python 3.16 将废弃 `asyncio.iscoroutinefunction`
   - 影响: 未来版本可能不兼容
   - 解决: 等待 pytest-asyncio 更新
   - 状态: ⏸️ 暂不处理
