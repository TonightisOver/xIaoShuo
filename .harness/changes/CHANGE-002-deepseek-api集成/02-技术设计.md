## 1. 设计概述

集成 DeepSeek API 到 LangGraph 节点中，使用 LangChain 的 ChatOpenAI 适配器连接 DeepSeek，实现异步调用和错误处理。


### 3.2 LLM 客户端

**文件**: `src/core/llm/client.py`

```python
from langchain_openai import ChatOpenAI
from tenacity import retry, stop_after_attempt, wait_exponential
from src.core.config import get_settings

class LLMClient:
    """LLM 客户端封装"""
    
    def __init__(self):
        settings = get_settings()
        self.llm = ChatOpenAI(
            model=settings.DEEPSEEK_MODEL,
            api_key=settings.DEEPSEEK_API_KEY,
            base_url=settings.DEEPSEEK_BASE_URL,
            temperature=settings.DEEPSEEK_TEMPERATURE,
            max_tokens=settings.DEEPSEEK_MAX_TOKENS,
            timeout=settings.DEEPSEEK_TIMEOUT,
        )
    
    @retry(
        stop=stop_after_attempt(3),
        wait=wait_exponential(multiplier=1, min=2, max=10)
    )
    async def generate(
        self,
        prompt: str,
        temperature: float | None = None,
        max_tokens: int | None = None
    ) -> str:
        """生成文本"""
        messages = [{"role": "user", "content": prompt}]
        
        kwargs = {}
        if temperature is not None:
            kwargs["temperature"] = temperature
        if max_tokens is not None:
            kwargs["max_tokens"] = max_tokens
        
        response = await self.llm.ainvoke(messages, **kwargs)
        return response.content

# 全局客户端实例
_client: LLMClient | None = None

def get_llm_client() -> LLMClient:
    """获取 LLM 客户端单例"""
    global _client
    if _client is None:
        _client = LLMClient()
    return _client
```

### 3.4 节点改造

#### 改造方案

1. **函数签名改为异步**
   ```python

   def node(state: NovelState) -> NovelState:
       pass
   
   # 改造后
   async def node(state: NovelState) -> NovelState:
       pass
   ```

2. **调用 LLM 生成内容**
   ```python
   from src.core.llm.client import get_llm_client
   from src.core.llm.prompts import IDEA_EXPANSION_PROMPT
   
   async def node(state: NovelState) -> NovelState:
       try:
           client = get_llm_client()
           prompt = IDEA_EXPANSION_PROMPT.format(
               idea=state["idea"],
               novel_type=state["novel_type"],
               target_words=state["target_words"]
           )
           result = await client.generate(prompt)
           return {**state, "idea": result}
       except Exception as e:
           # 降级到 mock 数据
           return {**state, "errors": state["errors"] + [str(e)]}
   ```

3. **保留降级方案**
   - API 调用失败时使用 mock 数据
   - 记录错误到 state["errors"]

## 5. 错误处理

### 5.1 错误类型

1. **网络错误**: 连接超时、网络中断
2. **API 错误**: 认证失败、配额超限、请求格式错误
3. **响应错误**: 解析失败、格式不符
4. **业务错误**: 生成内容质量差

### 5.2 处理策略

```python
try:
    # 尝试调用 API
    result = await client.generate(prompt)
except TimeoutError:
    # 超时 -> 重试
    logger.warning("API timeout, retrying...")
    raise  # tenacity 会自动重试
except AuthenticationError:
    # 认证失败 -> 记录错误，使用降级方案
    logger.error("API authentication failed")
    return fallback_result
except RateLimitError:
    # 配额超限 -> 等待后重试
    logger.warning("Rate limit exceeded, waiting...")
    await asyncio.sleep(60)
    raise
except Exception as e:
    # 其他错误 -> 记录错误，使用降级方案
    logger.error(f"API call failed: {e}")
    return fallback_result
```
