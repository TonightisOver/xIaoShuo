## 新增 API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/v1/novels/long-form | 创建百万字长篇生成任务 |
| GET | /api/v1/novels/{novel_id}/quality-report | 获取质量趋势报告 |
| GET | /api/v1/novels/{novel_id}/filler-detection | 获取注水检测结果 |
| GET | /api/v1/novels/{novel_id}/foreshadow-tracker | 获取伏笔追踪报告 |
| GET | /api/v1/novels/{novel_id}/progress | 获取百万字长篇进度 |
| POST | /api/v1/novels/{novel_id}/volumes/{volume_number}/generate | 按卷触发生成 |
| POST | /api/v1/novels/{novel_id}/volumes/{volume_number}/pause | 暂停指定卷生成 |
| POST | /api/v1/novels/{novel_id}/volumes/{volume_number}/resume | 恢复指定卷生成 |
