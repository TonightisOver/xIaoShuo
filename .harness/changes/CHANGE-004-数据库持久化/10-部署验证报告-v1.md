## 4. 回滚方案

如需回滚:

1. 停止应用
2. 执行 `alembic downgrade -1` 回退数据库
3. 恢复 task_manager.py 为 JSON 版本
4. 重启应用

JSON 文件作为备份保留，可随时恢复。
