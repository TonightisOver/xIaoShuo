# 编码计划: PostgreSQL 数据库持久化

**变更 ID**: CHANGE-004  
**创建时间**: 2026-05-16

## 1. 任务概述

将任务存储从 JSON 文件迁移到 PostgreSQL，实现 SQLAlchemy 2.0 async ORM、Alembic 迁移、TaskManager 重写、测试配置。

## 7. 风险和注意事项

- **风险 1**: pytest-asyncio 与 asyncpg 事件循环冲突
  - **缓解**: 使用 NullPool，已验证可行

- **风险 2**: Alembic 迁移与 async engine 不兼容
  - **缓解**: env.py 中使用同步连接执行迁移

- **注意事项 1**: .env 文件包含数据库密码，不要提交到 git
- **注意事项 2**: 测试数据库需要手动创建（CREATE DATABASE xiaoshuo_test）
- **注意事项 3**: 迁移脚本执行前确保 JSON 文件存在
