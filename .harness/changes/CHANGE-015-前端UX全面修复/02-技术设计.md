### 3.3 对话结论应用

结论列表每条建议后添加"应用"按钮：
- type=world → 调用 PUT /projects/{id}/world，追加到对应字段
- type=character → 调用 POST /projects/{id}/characters，创建新人物
- type=plot → 显示为参考（不自动应用）

### 3.4 Conversation 上下文修复

当前问题：world_setting 可能返回空对象（所有字段为 null），characters 只取了 name。

修复：
```python
# 读取完整世界观
if world:
    parts = []
    if world.get("background"): parts.append(f"背景：{world['background']}")
    if world.get("rules"): parts.append(f"规则：{world['rules']}")
    if world.get("culture"): parts.append(f"文化：{world['culture']}")
    context_parts.append("世界观：\n" + "\n".join(parts))

# 读取完整人物
for char in characters[:5]:
    char_info = f"{char['name']}({char.get('role', '未知')})"
    if char.get("description"): char_info += f" - {char['description']}"
    context_parts.append(f"人物：{char_info}")
```
