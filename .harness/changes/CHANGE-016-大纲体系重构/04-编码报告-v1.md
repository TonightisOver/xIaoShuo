## 实现要点

1. **统一 outlines 表**: level 字段区分 master/volume/chapter，一张表存三级
2. **LLM 生成**: 3 个生成方法（卷纲/章纲/从对话提取总纲）
3. **树形 API**: GET /outlines 返回完整树形结构（master + volumes + chapters）
4. **前端编辑器**: 总纲可编辑，卷纲/章纲可逐级生成
5. **从对话生成**: 对话结束后可一键提取为结构化总纲
