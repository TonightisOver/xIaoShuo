## C1. 抽取 `_build_initial_state`

**文件：** `src/api/services/novel_generator.py`

**新增函数签名：**
```python
async def _build_initial_state(
    task_id: str,
    request: CreateNovelRequest,
) -> dict[str, Any]:
    """构建 LangGraph 初始状态，注入已有设定。

    从 task 关联的 novel 中获取 world_setting、characters、storylines，
    合并到初始状态中。
    """
```

**函数体逻辑：**
```python
async def _build_initial_state(
    task_id: str,
    request: CreateNovelRequest,
) -> dict[str, Any]:
    initial_state: dict[str, Any] = {
        "project_id": task_id,
        "idea": request.idea,
        "novel_type": request.novel_type,
        "target_words": request.target_words,
        "writing_style": request.writing_style,
        "writing_style_prompt": request.writing_style_prompt,
        "current_stage": "start",
        "chapters": [],
        "errors": [],
    }

    task_manager = get_task_manager()
    task_data = await task_manager.get_task(task_id)
    novel_id = task_data.get("novel_id") if task_data else None
    if novel_id:
        novel_manager = get_novel_manager()
        existing_world = await novel_manager.get_world_setting(novel_id)
        existing_chars = await novel_manager.list_characters(novel_id)
        ws_keys = ["background", "rules", "culture", "geography"]
        if existing_world and any(existing_world.get(k) for k in ws_keys):
            initial_state["world_setting"] = existing_world
        if existing_chars:
            initial_state["characters"] = existing_chars
        from src.api.services.storyline_service import get_storyline_service
        sl_service = get_storyline_service()
        storylines = await sl_service.list_storylines(novel_id)
        if storylines:
            sl_parts = [
                f"- [{sl['type']}] {sl['name']}: {sl.get('description', '')}"
                for sl in storylines
            ]
            sl_context = "\n".join(sl_parts)
            initial_state["idea"] = f"{request.idea}\n\n已确定的故事线：\n{sl_context}"

    return initial_state
```

**调用方修改：**
- `generate_novel_background`：删除第 84-119 行，替换为 `initial_state = await _build_initial_state(task_id, request)`
- `generate_novel_full_background`：删除第 314-347 行，替换为 `initial_state = await _build_initial_state(task_id, request)`

**返回值额外用途：** `generate_novel_full_background` 在调用后还需要 `novel_id` 用于 persist，可从 task_data 再次获取（已有逻辑），或让函数返回 tuple `(initial_state, novel_id)`。

**推荐方案：** 返回 tuple：
```python
async def _build_initial_state(
    task_id: str,
    request: CreateNovelRequest,
) -> tuple[dict[str, Any], str | None]:
    """返回 (initial_state, novel_id)"""
    ...
    return initial_state, novel_id
```

## H4. generate_novel_background 复用 _run_langgraph_pipeline

**文件：** `src/api/services/novel_generator.py`

**当前问题：** `generate_novel_background`（第 120-186 行）内联了 graph streaming + progress 逻辑，而 `_run_langgraph_pipeline` 已经封装了相同逻辑。

**修改方案：**

将 `generate_novel_background` 的第 121-164 行（register callback → graph.astream 循环 → complete_task → persist）替换为：

```python
    initial_state, novel_id = await _build_initial_state(task_id, request)

    await event_bus.publish(ProgressEvent(
        task_id=task_id,
        event_type=EventType.STAGE_START,
        data={"stage": "idea_expansion", "percentage": 0},
    ))

    result = await _run_langgraph_pipeline(task_id, initial_state, stage_offset=0)

    await task_manager.complete_task(task_id, result)

    if novel_id:
        await _persist_to_novel(novel_id, result)

    await event_bus.publish(ProgressEvent(
        task_id=task_id,
        event_type=EventType.COMPLETED,
        data={"percentage": 100, "current_stage": "completed"},
    ))
```

**百分比差异处理：** `_run_langgraph_pipeline` 使用 `_full_generate_percentage` 计算百分比（基于 13 阶段）。当 `generate_novel_background` 调用时，需要基于 7 阶段计算。

**解决方案：** 给 `_run_langgraph_pipeline` 增加可选参数 `total_stages: int = 13`：
```python
async def _run_langgraph_pipeline(
    task_id: str,
    initial_state: dict[str, Any],
    stage_offset: int = 0,
    total_stages: int | None = None,
) -> dict[str, Any]:
```

当 `total_stages` 为 None 时使用 `len(FULL_GENERATE_STAGES)`，否则用传入值。`generate_novel_background` 调用时传 `total_stages=7`。

**函数体：**
```python
async def _generate_single_chapter(
    client: Any,
    chapter_outline: dict[str, Any],
    previous_chapter: str,
    characters_json: str,
    world_setting_json: str,
    storylines_json: str = "",
    style_instruction: str = "",
    kg_service: Any | None = None,
    novel_id: str | None = None,
) -> dict[str, Any]:
    import json
    from src.core.llm.prompts import CHAPTER_GENERATION_PROMPT

    # 知识图谱上下文（可选）
    kg_context = ""
    if kg_service and novel_id:
        try:
            kg_context = await kg_service.retrieve_context(
                novel_id=novel_id,
                chapter_outline=chapter_outline,
            )
        except Exception as e:
            logger.warning("kg_context_retrieval_failed", error=str(e))

    prompt = CHAPTER_GENERATION_PROMPT.format(
        chapter_outline=json.dumps(chapter_outline, ensure_ascii=False),
        previous_chapter=previous_chapter or "这是第一章",
        characters=characters_json,
        world_setting=world_setting_json,
    )

    if kg_context:
        prompt = f"{kg_context}\n\n{prompt}"
    if storylines_json:
        prompt += f"\n\n已确定的故事线：\n{storylines_json}"
    if style_instruction:
        prompt = f"{style_instruction}\n\n{prompt}"

    content = await client.generate(prompt, max_tokens=8000)

    # 知识抽取（可选）
    if kg_service and novel_id:
        try:
            await kg_service.extract_from_chapter(
                novel_id=novel_id,
                chapter_number=chapter_outline.get("chapter", 0),
                chapter_text=content,
            )
        except Exception as e:
            logger.warning("kg_extraction_failed", error=str(e))

    return {
        "chapter": chapter_outline.get("chapter", 0),
        "title": chapter_outline.get("title", ""),
        "content": content,
        "word_count": len(content),
    }
```

**调用方修改：**

1. **`chapter_generation.py` 节点：** 循环体替换为调用 `_generate_single_chapter`，传入 `kg_service` 和 `novel_id`
2. **`generate_volume_background`：** 第 640-660 行替换为调用，不传 `kg_service`
3. **`generate_chapters_background`：** 第 783-802 行替换为调用，不传 `kg_service`

**导入路径：** LangGraph 节点需要从 `src.api.services.novel_generator` 导入此函数，或将其放在独立模块 `src/core/chapter_utils.py` 中避免循环依赖。

**推荐：** 放在 `src/core/llm/chapter_generator.py` 中，因为它本质是 LLM 调用逻辑，不依赖 service 层。
