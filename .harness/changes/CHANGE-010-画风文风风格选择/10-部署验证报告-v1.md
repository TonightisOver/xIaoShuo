# 部署验证报告 v1

## 变更编号
CHANGE-010

## 部署环境
- 服务器: Docker 容器化部署
- 数据库: PostgreSQL 17
- 部署方式: docker-compose 重新构建

## 部署步骤

1. 拉取最新代码
2. `docker-compose build api` — 重新构建 API 镜像
3. `docker-compose up -d` — 重启服务
4. 容器内自动执行 `alembic upgrade head`

## 回滚方案

```bash
# 回滚迁移
docker-compose exec api alembic downgrade -1
# 重新部署旧版本
docker-compose build api && docker-compose up -d
```
