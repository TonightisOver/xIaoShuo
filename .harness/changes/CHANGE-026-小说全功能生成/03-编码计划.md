## 关键设计决策

1. **Wrapper 模式**: 不修改现有 `generate_novel_background()`，新增独立函数，复用 `_persist_to_novel()`
2. **子功能串行执行**: 每步独立 try-catch + WebSocket 进度推送，失败不阻断后续
3. **13 阶段计算**: 每个阶段约 7.7% 进度（100/13）
4. **LangGraph pipeline 提取**: `_run_langgraph_pipeline()` 提取公共逻辑，被 old/new 两个函数复用
5. **前端路径匹配**: API 调用路径与后端路由精确匹配
