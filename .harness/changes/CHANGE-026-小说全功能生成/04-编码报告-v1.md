# 编码报告 -- CHANGE-026-小说全功能生成

**日期**: 2026-05-18
**版本**: v1

## 执行摘要

完成了"一键全功能生成"13 阶段流水线的全部编码实现。核心改动包括：

1. 新增 `generate_novel_full_background()` wrapper 函数，在现有 LangGraph 7 节点流程基础上串联 6 个额外 AI 生成步骤
2. 提取 `_run_langgraph_pipeline()` 辅助函数复用 LangGraph 运行逻辑
3. 新增 `_run_sub_feature()` 辅助函数统一子功能的进度事件和错误处理
4. 各 Service 层新增对应方法（力量体系、大纲持久化、自动对话）
5. 前端新增"一键全功能生成"入口和对应的进度展示

## 关键实现细节

### 后端

1. **13 阶段流水线**: `generate_novel_full_background()` 按顺序执行 LangGraph 7 阶段 + 6 个子功能
2. **错误隔离**: 每个子功能通过 `_run_sub_feature()` 独立 try-catch，失败推送 `ERROR` 事件（`non_blocking: true`），不阻断后续
3. **进度推送**: 每个阶段通过 `SUB_FEATURE_START` / `SUB_FEATURE_COMPLETE` EventType 推送进度
4. **复用现有函数**: 子功能调用已有的 `generate_storylines_ai()`, `generate_arcs_ai()`, `generate_scenes_ai()`, `generate_master_from_novel()`

### 前端

1. **Create.vue**: 新增 `fullGenerate()` 调用 `/api/v1/projects/full-generate` 并跳转到 TaskDetail
2. **NovelDetail.vue**: 新增 `fullGenerate()` 调用 `/api/v1/projects/{novel_id}/generate-full`, 新增 power-systems tab 展示力量体系, outlines tab 展示大纲树, storylines tab 展示故事线/弧光/场景
3. **TaskDetail.vue**: WebSocket 消息处理新增 `sub_feature_start` / `sub_feature_complete` 事件类型
4. **StageIndicator.vue**: stages 数组从 7 扩展到 13
